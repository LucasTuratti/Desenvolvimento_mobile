# 📱 Calculadora de IMC — Kotlin

Exercício de desenvolvimento mobile desenvolvido em **Kotlin** no **Android Studio**, 
aplicando conceitos de **Programação Orientada a Objetos**.

---

## 📋 Sobre o Projeto

O programa calcula e classifica o **Índice de Massa Corporal (IMC)** de duas pessoas 
a partir dos dados inseridos pelo usuário: nome, idade, peso e altura.

---

## ⚙️ Funcionalidades

- Cadastro de duas pessoas com nome, idade, peso e altura
- Cálculo automático do IMC
- Classificação do IMC seguindo a tabela oficial:

| IMC | Classificação |
|-----|--------------|
| Abaixo de 18.5 | Abaixo do Peso Ideal |
| 18.5 – 24.9 | Peso Normal |
| 25.0 – 29.9 | Acima do Peso |
| 30.0 – 34.9 | Obesidade Grau 1 |
| 35.0 – 39.9 | Obesidade Grau 2 |
| Acima de 40.0 | Obesidade Grau 3 |
